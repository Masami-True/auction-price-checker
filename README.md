# オークション仕入れ 価格チェッカー

出品商品詳細PDFをアップロードすると、各サイトの市場価格を自動取得して表示するWebアプリです。

## ローカルで動かす方法

```bash
# 1. Node.js (v18以上) をインストール済みであること

# 2. 依存パッケージをインストール
npm install

# 3. サーバーを起動
npm start

# 4. ブラウザで開く
# http://localhost:3000
```

## Renderにデプロイする方法

1. このフォルダをGitHubリポジトリにpush
2. [Render](https://render.com) にサインアップ（無料）
3. 「New Web Service」→ GitHubリポジトリを選択
4. 自動的に `render.yaml` の設定が読み込まれる
5. 「Create Web Service」をクリック
6. 数分後にURLが発行される → そのURLを代行会社さんに共有

## 機能

- PDF解析（商品名、ブランド、ライン、型番、評価、価格を自動抽出）
- 画像抽出（PDFから商品写真を取り出してGoogle Lens検索）
- 価格自動取得（メルカリ、ヤフオク、楽天、オークフリー、Googleショッピング）
- 最安値・最高値・平均・中央値をサマリー表示
- 手動検索リンク（オークファン含む各サイトへ直リンク）
